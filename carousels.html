Responsive Carousel

OWL Carousel is touch enabled jQuery plugin that lets you create beautiful responsive carousel sliders. You can find more detailed information from the offecial plugin website http://owlgraphic.com/owlcarousel

Default example

For default carousel you should use a class .owl-carousel-v1. Moreover, you are also able to use it without navigation. You should just remove owl-navigation block from the HTML code.

Centrered navigation with fixed width image

For centered navigation you should change classs .owl-carousel-v1 to .owl-carousel-v2. You should add an additional class .owl-carousel-style-v1 to make fixed width image style.


Classic style and auto play

You should just add an additional class .owl-carousel-style-v2 to make classic style. For auto play you may use .owl-slider-v3 class.


Image carousel

You are also able to use the carousel as a blcok or image slider.
